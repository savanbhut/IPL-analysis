# IPL Match Analysis

## Overview
Exploratory data analysis on IPL match data (2008–2022).
Dataset: ~900 matches, 180,000+ ball-by-ball deliveries.

## Questions answered
- Which teams won the most matches?
- Who are the top run scorers and wicket takers?
- Does winning the toss impact match result?
- How have total runs trended season by season?

## Tools used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key findings
- Mumbai Indians have the most IPL wins overall
- Toss win leads to match win only ~51% of the time
- Run scoring has increased significantly post-2017
